Project 01 – Active Directory Administration

Overview
This project documents the deployment and administration of an Active Directory environment using Windows Server 2022 in a VMware-based cybersecurity homelab.

Objectives
- Deploy a Domain Controller
- Create the BENLAB.LOCAL domain
- Configure DNS
- Create Organizational Units
- Create users and security groups
- Configure password and account lockout policies
- Join Windows workstations to the domain

Technologies
- Windows Server 2022
- Windows 11
- VMware Workstation Pro
- Active Directory Domain Services
- DNS
- Group Policy
- PowerShell

Skills Demonstrated

- Active Directory
- User and Group Management
- DNS Administration
